TryParseExact Method (value, patterns, formatProvider, result)
Namespaces ► NodaTime ► LocalDate ► TryParseExact(String, array<String>[]()[][], IFormatProvider, LocalDate%)

[This is preliminary documentation and is subject to change.]

Attempts to parse the given string using the specified patterns and format provider. If the parse is successful, the result is stored in the result parameter and the return value is true; otherwise January 1st, 2000 is stored in the parameter and the return value is false.
