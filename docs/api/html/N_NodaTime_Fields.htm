The NodaTime.Fields namespace contains types related to individual fields within date and time types. Most types within this namespace are internal; most users will not need to use the public types here.

Structures

  StructureDescription
PeriodFieldValue
A simple combination of a PeriodFieldType and a 64-bit integer value. These are used when representing periods.

Enumerations

  EnumerationDescription
PeriodFieldType
Indicates the type of a value represented by a period field.
