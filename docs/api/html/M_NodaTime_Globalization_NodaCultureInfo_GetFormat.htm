Parameters
formatType (Type)
The Type for which to get a formatting object. This method only supports the NumberFormatInfo and DateTimeFormatInfo types.
Return Value
The value of the NumberFormat property, which is a NumberFormatInfo containing the default number format information for the current CultureInfo, if formatType is the Type object for the NumberFormatInfo class. -or- The value of the DateTimeFormat property, which is a DateTimeFormatInfo containing the default date and time format information for the current CultureInfo, if formatType is the Type object for the DateTimeFormatInfo class. -or- null, if formatType is any other object.
