TryParse(String, LocalDateTime%)
Attempts to parse the given string using the current culture's default format provider. If the parse is successful, the result is stored in the result parameter and the return value is true; otherwise DefaultTemplateValue()()()() is stored in the parameter and the return value is false.

TryParse(String, IFormatProvider, LocalDateTime%)
Attempts to parse the given string using the specified format provider. If the parse is successful, the result is stored in the result parameter and the return value is true; otherwise DefaultTemplateValue()()()() is stored in the parameter and the return value is false.
